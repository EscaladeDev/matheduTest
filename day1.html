Classifying a Real Number

CLEP quiz overview

Diagnostic test: CLEP-style multiple-choice questions that cover the scope of OpenStax College Algebra 2e Section 1.1 up to and including the subsection “Classifying a Real Number”.

CLEP quiz focus

  • The hierarchy of real numbers.
  • Definitions and examples of natural numbers (counting: 1, 2, 3, …), whole numbers (natural + 0), integers (whole + negatives), rational numbers (quotients of integers, m/n where n ≠ 0, including terminating/repeating decimals), irrational numbers (non-repeating, non-terminating decimals; cannot be written as m/n).
  • The subset relationships (ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ, irrationals ⊂ ℝ).
  • Classifying a given number into one or more categories and determining if it is positive/negative and its position relative to zero on the number line.
  • Recognizing terminating vs. repeating decimals for rationals.
  • Basic conceptual questions about why certain numbers fit (or don't fit) each category.

CLEP quiz strategy

Question design

The questions mimic CLEP College Algebra style: mostly 4 options, blend of direct classification, reasoning about why a number belongs in a set, common traps (e.g., misclassifying 0 or √4), and conceptual items (e.g., subset questions, decimal form implications). They stay strictly within the “Classifying a Real Number” scope—no order of operations, properties, or algebraic simplification.

Classifying Real Numbers

Step-by-Step Solutions

Which of the following numbers is a natural number? A. 0 B. −5 C. 7 D. 1/2

Correct answer: C (7)

  • Natural numbers are the counting numbers starting from 1: 1, 2, 3, 4, …
  • 0 is a whole number but not natural (whole numbers = natural numbers + 0).
  • −5 is negative → integer but not natural.
  • 1/2 is a fraction → rational but not natural.
  • Only 7 fits the definition of natural numbers. Trap: Many confuse natural numbers with whole numbers (including 0).

The number 0 belongs to which of the following sets? (Select the most complete correct choice.) A. Natural numbers only B. Whole numbers and integers C. Integers and rationals D. Whole numbers, integers, and rationals

Correct answer: D

  • Whole numbers: 0, 1, 2, 3, … → includes 0.
  • Integers: …, −2, −1, 0, 1, 2, … → includes 0.
  • Rationals: any number m/n where m, n are integers, n ≠ 0 → 0 = 0/1.
  • 0 is not natural (natural starts at 1).
  • Most complete: whole, integer, and rational. Trap: Choice C omits whole numbers.

Which number is an integer but not a whole number? A. 4 B. 0 C. −3 D. 2.5

Correct answer: C (−3)

  • Whole numbers are non-negative integers: 0, 1, 2, 3, …
  • Integers include negatives: …, −3, −2, −1, 0, 1, 2, …
  • −3 is an integer but negative, so it is not whole.
  • 4 and 0 are whole numbers.
  • 2.5 is not an integer. Trap: Negatives are integers, but not whole numbers.

√9 is an example of a(n): A. Irrational number B. Rational number that is not an integer C. Integer that is not a whole number D. Integer and rational number

Correct answer: D

  • √9 = 3, a perfect square.
  • 3 is a natural number → whole number → integer → rational (3/1).
  • So 3 is both integer and rational.
  • Trap: Choice C is impossible here because 3 is whole.

Which of the following is an irrational number? A. −4/5 B. 0.75 C. √25 D. √7

Correct answer: D (√7)

  • Irrational numbers cannot be written as m/n and have non-repeating, non-terminating decimals.
  • −4/5 = −0.8 → rational.
  • 0.75 = 3/4 → rational and terminating.
  • √25 = 5 → rational.
  • √7 ≈ 2.64575… → non-repeating, non-terminating → irrational.

The decimal 0.3333… (repeating) is A. Irrational B. Rational and terminating C. Rational and repeating D. An integer

Correct answer: C

  • 0.333… = 1/3.
  • All repeating decimals are rational because they can be written as fractions.
  • It is not terminating and not an integer.

The decimal form of √2 is approximately 1.414213562… This decimal is A. Terminating B. Repeating C. Non-repeating and non-terminating D. Rational

Correct answer: C

  • Irrational numbers have decimal expansions that are non-repeating and non-terminating.
  • √2 is irrational, so its decimal never repeats and never stops.
  • Trap: It is not rational.

Which number is rational but not an integer? A. −8 B. 0 C. 3/4 D. √4

Correct answer: C (3/4)

  • Rational numbers can be written as m/n where n ≠ 0.
  • −8 = −8/1 and 0 = 0/1, so both are integers and rational.
  • 3/4 = 0.75 → rational, not an integer.
  • √4 = 2 → integer and rational.

On the number line, where would the number −√2 lie relative to 0? A. To the right of 0 (positive) B. To the left of 0 (negative) C. At 0 D. Cannot be placed (not real)

Correct answer: B

  • √2 ≈ 1.414, which is positive.
  • Therefore −√2 ≈ −1.414, which lies to the left of 0.
  • It is real: irrational but still part of the real numbers.

Which set is the most specific classification for the number −17? A. Rational numbers B. Integers C. Whole numbers D. Natural numbers

Correct answer: B

  • −17 is negative, so it is not whole and not natural.
  • It is an integer.
  • It is also rational (−17/1), but integer is the more specific classification.

The number 0.1010010001… (non-repeating pattern of increasing zeros) is A. Rational B. Irrational C. Integer D. Whole number

Correct answer: B

  • Rational decimals are either terminating or repeating.
  • This pattern never repeats and never terminates, so it is irrational.
  • Trap: A visible pattern does not automatically make it rational.

Which of the following statements is true? A. All whole numbers are natural numbers. B. All integers are whole numbers. C. All rational numbers are integers. D. All natural numbers are integers.

Correct answer: D

  • Natural ⊂ whole ⊂ integers ⊂ rationals ⊂ reals.
  • Only D is true.
  • A is false because whole numbers include 0.
  • B is false because integers include negatives.
  • C is false because rationals include fractions.

Is 15/5 a rational number? If so, is it terminating or repeating in decimal form? A. Irrational B. Rational and terminating C. Rational and repeating D. Not a real number

Correct answer: B

  • 15/5 = 3, which is an integer.
  • 3.0 is terminating.
  • All integers are rational and have terminating decimals.

Which number lies to the left of 0 on the number line and is rational? A. √5 B. −0.8 C. 0 D. 2/3

Correct answer: B (−0.8)

  • Left of 0 means negative.
  • −0.8 = −4/5, which is rational.
  • √5 and 2/3 are positive, and 0 is at zero.

The number π (approximately 3.14159…) is classified as A. Rational and repeating B. Irrational C. Integer D. Whole number

Correct answer: B

  • π is the classic irrational number.
  • Its decimal is non-repeating and non-terminating.
  • It is not rational, not an integer, and not whole.

Which of the following is NOT a rational number? A. −7/1 B. 0/5 C. √16 D. √10

Correct answer: D (√10)

  • −7/1 = −7 → rational.
  • 0/5 = 0 → rational.
  • √16 = 4 → rational.
  • √10 cannot be written as a fraction of integers, so it is irrational.

True or false: Every integer is a rational number. A. True B. False C. Only positive integers D. Only negative integers

Correct answer: A (True)

  • Any integer n can be written as n/1.
  • Therefore every integer is rational.

The number −3.0 is A. Irrational B. Rational, terminating decimal C. Whole number D. Natural number

Correct answer: B

  • −3.0 = −3 = −3/1, so it is rational.
  • The decimal terminates.
  • Because it is negative, it is not whole or natural.

Which number is both negative and irrational? A. −4/7 B. −√3 C. −0.25 D. −8

Correct answer: B (−√3)

  • −4/7, −0.25, and −8 are rational.
  • √3 is irrational, so −√3 is negative and irrational.

If a number can be written as a fraction m/n where m and n are integers and n ≠ 0, it is A. Irrational B. Rational C. Only an integer D. Only a whole number

Correct answer: B (Rational)

  • This is the exact definition of a rational number.
  • Integers and whole numbers are subsets of rationals, but not all rationals are integers or whole.
