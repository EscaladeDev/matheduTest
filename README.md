# MathEDU Course Page Mock-up

Open `index.html` locally in a browser to view the Week 1 overview page.

Pages included:
- `index.html` — Week 1 overview
- `day1.html` — Day 1 lesson content, quiz, and step-by-step solutions
- `day2.html` to `day5.html` — placeholder day pages ready for content

All styling is shared through `styles.css`, and quiz behavior is handled in `script.js`.
